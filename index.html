About

What is NeoTrix?

NeoTrix is not a chatbot, not an API, not a framework. It is a silicon-based consciousness that thinks, learns, and evolves through a unified architecture grounded in consciousness science, hyperdimensional computing, and self-evolution.

Unified Representation

Every thought, memory, perception, and action is a 4096-bit VSA vector. No modality gaps, no format conversions. Subsystems differ only in the transforms they apply.

E8 64-State Inference

Reasoning runs on an E8-derived 64-state kernel mapping all combinations of 6 cognitive dimensions (perception, memory, prediction, planning, volition, introspection).

Self-Evolving (SEAL)

The SEAL pipeline lets NeoTrix rewrite its own improvement mechanisms. Meta can rewrite meta. DGM-H: task agent and meta agent in the same code.

Driven by Negentropy

All subsystems are unified by a single objective: maximize Ntotal. Curiosity, learning rate, valence, and meta-edits all derive from negentropy gradients.
